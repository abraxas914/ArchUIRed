## Overview

The selected state adds a 2px blue border (`accent/primary`) and a glow ring shadow (`0 0 0 3px accent/primary` at 20% opacity) to the card. The background is unchanged from the default state. The drill-in icon becomes permanently visible rather than hover-only.

Visual specification with annotated diagram is in `resources/visual-spec.md`.
