## 概述

模块广场是 ArchUI 区别于所有竞品的核心差异化功能。用户可以浏览社区共享的文档模块，按领域和样式选择，一键拉取或替换到自己的项目中。

## 核心功能

| 功能 | 说明 |
|---|---|
| 模块卡片网格 | 按领域/标签分类展示（登录、支付、鉴权、仪表盘、落地页、UI 组件…） |
| 搜索 & 筛选 | 关键词搜索、标签筛选、排序（热度/最新/评分/下载量） |
| 模块预览页 | 模块树结构可视化 + 效果截图 + README 内容 + 评分评论 |
| 样式预览对比 | 同类模块并排对比（如：4 种不同风格的登录模块） |

## 样式作为卖点

同一功能、多种样式是广场的杀手级场景：

| 模块类型 | 样式变体示例 |
|---|---|
| 登录页面 | 极简白、渐变玻璃、暗黑风、暖色自然 |
| 仪表盘 | 卡片式、暗色数据面板 |
| 落地页 | SaaS 风、大胆暗色 |
| 设置页面 | iOS 风格、Material Design |

## 发布模块

用户可以将自己的模块发布到广场：选中本地模块树 → 填写标签/分类/截图 → 一键发布。支持版本迭代更新，引用方可选择是否升级。

## 广场 Registry 初期方案

初期使用 GitHub repo 作为 registry，`registry.json` 存储模块索引（名称、标签、描述、预览图 URL）。后期迁移到独立服务。
