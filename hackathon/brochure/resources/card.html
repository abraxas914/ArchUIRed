BACK — 反面
核心亮点
1
一键 Skills 化
将知识文档变成可复用、相关联的 Skills 模块。
一键导入现有项目，自动生成模块树。
archui init → 扫描文档 → 生成 Skills 模块树
2
文档即编排
编排知识模块就是编排代码。
改 Skills 即改代码，AI 自动同步更新。
edit SPEC.md → AI auto-generates code
3
模块广场
发布、发现、替换、组合 — Skills 模块的 npm。
社区共享知识模块，构建复用生态。
publish → discover → replace → generate
