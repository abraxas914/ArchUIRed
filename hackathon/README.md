## 概述

Hackathon 模块管理参加黑客松活动所需的全部展示和传播材料。每个子模块负责一类输出物，从视觉设计到内容叙事，形成完整的项目展示体系。

## 子模块

| 模块 | 职责 |
|---|---|
| `basic-info` | 基础信息（队名、项目名称、一句话项目介绍的统一来源） |
| `ppt` | 演示文稿（幻灯片结构、内容、设计规范） |
| `banner` | 易拉宝（展架设计规范与内容布局） |
| `video` | 演示视频（脚本、分镜、剪辑规范） |
| `memory-nodes` | 项目记忆节点（关键决策、洞察与叙事锚点） |
| `project-packaging` | 项目包装（问题陈述、竞品对比、市场分析） |
| `brochure` | 宣传册（折页设计与内容规范） |
| `image-design` | 图片生成式设计（基于 Gemini 的视觉素材生成） |
| `vi` | 视觉识别系统（统一管控全套材料的品牌风格） |
| `web-demo` | Web 版本 Demo 规划（画布编排 + 模块广场 + 一键替换） |
