## 概述

易拉宝模块负责黑客松现场竖版展架（Roll-up Banner）的设计规范，适用于 80cm × 200cm 标准尺寸。

## 内容分区（从上到下）

| 区域 | 高度占比 | 内容 |
|---|---|---|
| **品牌顶栏** | 10% | Arch Park + Logo |
| **主视觉** | 25% | 节点画布截图 / 模块连线图 |
| **产品定位** | 10% | AI-Native Generative Knowledge Engine / AI 原生生成式知识引擎 |
| **项目介绍** | 15% | 价值主张 + 痛点 + 解决方案 |
| **三大亮点** | 25% | 图标 + 短句，三列排布 |
| **二维码 / 团队** | 15% | Demo 链接、团队信息 |

### 项目介绍文案

**价值主张：**
> 让工作知识可维护、可复用、可生成，可持续产生价值。

**痛点：**
> 项目知识层越来越复杂，难以维护。知识散落在代码和文档中，越积越乱，AI 也读不动。

**解决方案：**
> ArchUI 将知识结构化为可复用模块 — 每个模块独立可维护，AI 按需加载，直接生成代码。

### 三大亮点文案

| 亮点 | 标题 | 副文案 |
|---|---|---|
| 1 | **一键 Skills 化** | 将知识文档变成可复用、相关联的 Skills 模块 |
| 2 | **文档即编排** | 编排知识模块就是编排代码，改 Skills 即改代码 |
| 3 | **模块广场** | 发布、发现、替换、组合 — Skills 模块的 npm |

## 制作规范

- 尺寸：800mm × 2000mm，300 DPI 印刷输出（24000px × 60000px）
- 出血：四边各留 3mm
- 字体大小：标题 ≥ 120pt，正文 ≥ 48pt（1.5m 外可读）
- 主色调与 `brochure` 保持一致
- 主视觉图片来自 `image-design` 模块
