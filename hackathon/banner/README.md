## 概述

易拉宝模块负责黑客松现场竖版展架（Roll-up Banner）的设计规范，适用于 80cm × 200cm 标准尺寸。

## 内容分区（从上到下）

| 区域 | 高度占比 | 内容 |
|---|---|---|
| **品牌顶栏** | 10% | Arch Park + Logo |
| **主视觉** | 25% | 节点画布截图 / 模块连线图 |
| **项目名称 + 定位** | 10% | ArchUI — 模块化文档编排系统 / AI 开发基础设施 |
| **项目介绍** | 15% | 一段话说清楚 ArchUI 是什么、解决什么问题 |
| **三大亮点** | 25% | 图标 + 短句，三列排布 |
| **二维码 / 团队** | 15% | Demo 链接、团队信息 |

### 项目介绍文案

> 代码和文档是割裂的 — 文档写完就过时，AI 啃长文档效率低，知识无法复用。
>
> ArchUI 把文档拆成模块树，每个模块独立可理解、可复用。人类用画布编排模块关系，AI agent 按需加载上下文，直接生成代码。
>
> 文件系统即知识库，模块即 spec，文档改了代码跟着更新。

### 三大亮点文案

| 亮点 | 标题 | 副文案 |
|---|---|---|
| 1 | **文档即编排** | 文档模块化，AI 按需加载，告别长文档 |
| 2 | **模块可复用** | 拉取一个登录模块，AI 直接生成代码 |
| 3 | **模块广场** | 发布、发现、组合 — 知识模块的 npm |

## 制作规范

- 尺寸：800mm × 2000mm，300 DPI 印刷输出（24000px × 60000px）
- 出血：四边各留 3mm
- 字体大小：标题 ≥ 120pt，正文 ≥ 48pt（1.5m 外可读）
- 主色调与 `brochure` 保持一致
- 主视觉图片来自 `image-design` 模块
